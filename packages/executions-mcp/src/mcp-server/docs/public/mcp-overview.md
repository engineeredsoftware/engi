## 🚀 Getting Started

### **Quick Integration**

#### **Claude Desktop Integration**
```json
{
  "mcpServers": {
    "engi-engineering": {
      "command": "npx",
      "args": ["@bitcode/mcp-server"],
      "env": {
        "BITCODE_API_KEY": "your-api-key"
      }
    }
  }
}
```

#### **VS Code Extension**
```bash
# Install Bitcode MCP Extension
code --install-extension engi.mcp-integration

# Configure with your API key
code --command "engi.configure"
```

#### **API Integration**
```javascript
import { BitcodeMCPClient } from '@bitcode/mcp-client';

const client = new BitcodeMCPClient({
  apiKey: process.env.BITCODE_API_KEY,
  organizationId: 'your-org-id'
});

// Create a complete feature
const result = await client.createDeliverable({
  task: "Build user authentication system",
  repository: { owner: "myorg", name: "myapp" },
  options: { createPR: true, runTests: true }
});
```

### **Enterprise Deployment**

#### **Self-Hosted Option**
```bash
# Deploy to your infrastructure
docker run -d \
  --name bitcode-mcp \
  -e BITCODE_LICENSE_KEY=your-license \
  -e DATABASE_URL=your-database \
  -p 3000:3000 \
  engi/mcp-server:latest
```

#### **Cloud Deployment**
```yaml
# Kubernetes deployment
apiVersion: apps/v1
kind: Deployment
metadata:
  name: bitcode-mcp
spec:
  replicas: 3
  selector:
    matchLabels:
      app: bitcode-mcp
  template:
    spec:
      containers:
      - name: bitcode-mcp
        image: engi/mcp-server:latest
        env:
        - name: BITCODE_LICENSE_KEY
          valueFrom:
            secretKeyRef:
              name: engi-secrets
              key: license-key
```
